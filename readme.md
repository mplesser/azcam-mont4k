# azcam-mont4k

## Purpose

This repository contains the *azcam-bok* package.  It has the environement, configuration, and system files for the University of Arizona Kuiper telescope Mont4k facility camera system.

## Installation Example

Install only in editable mode because some files (such as DSP code) have static filenames. For Windows systems this code is usually cloned into `c:\azcam\azcam-mont4k`.

* `pip install -e .`
